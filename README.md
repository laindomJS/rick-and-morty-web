# Rick and Morty Web

A simple web that retrieves data from the Rick and Morty API and displays it dynamically.
Includes character finder and light/dark mode

## Tech Stack

**Client:** Svelte, TailwindCSS

**API:** [Rick&Morty API](rickandmortyapi.com)
